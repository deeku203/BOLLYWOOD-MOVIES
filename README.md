# BOLLYWOOD-MOVIES
This project is an end-to-end Power BI dashboard created to analyze Bollywood movie data, uncover trends, and provide actionable insights related to movie performance, cast, crew, budgets, revenues, and audience preferences.
The dashboard presents a clean, interactive, and visually appealing way to explore data using slicers, filters, KPIs, and charts.

Provide a dashboard useful for industry insights, business decisions, and portfolio analytics.
